Anotaciones sobre django rest framework y creaciones de API'S.

Django rest framework es una herramienta de django que te permite crear API'S.
Al no permitir que dos fuentes se conecten entre sí, la herramienta cors headers
nos permite determinar quién tendrá acceso a nuestra API.

para instalar django:

-pip install django

para instalar django rest framework:

-pip install djangorestframework

para instalar corsheaders:

-pip install django-cors-headers

para documentar la API:

-pip install coreapi

React:

React es una herramienta del frontend que nos permite desarrollar la web desde dicho lado.
Para instalarlo, primero debemos instalar node.js desde nuestro navegador. Es un archivo .exe.
luego, en la consola, ejecutamos los siguientes comandos:

(cabe aclarar que podemos hacerlo en otra consola mientras se ejecuta nuestra API de Django Rest Framework)

-npm create vite

vitejs es una herramienta de frontend muy rápida.

Luego, nos aparecerá el siguiente mensaje en la consola:

-√ Project name: ... client (colocamos el nombre del proyecto que en este caso será "client")

Luego nos mandará a elegir la herramienta de frontend que usaremos. Seleccionamos la que vamos a usar (React)

? Select a framework: » - Use arrow-keys. Return to submit.
>   Vanilla
    Vue
    React
    Preact

Desde la consola entramos a la carpeta creada (client):

-cd client

y ejecutamos el siguiente comando de nodejs:

-npm install

para correr el servidor, usamos este comando:

-npm run dev

Instalamos algunos módulos más para determinadas acciones:

-npm i react-router-dom react hot-toast axios react-hook-form
